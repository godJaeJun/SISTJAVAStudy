■■■ 음의 정수 표현 ■■■
1. -10을 byte 기반의 비트열로 나타낸다.
10 -> 00001010 -> 11110101 -> 11110110
2. -27을 byte 기반의 비트열로 나타낸다.
27 -> 00011011 -> 11100100 -> 11100101
3. -31을 byte 기반의 비트열로 나타낸다.
31 -> 00011111 -> 11100000 -> 11100001
4. -13을 byte 기반의 비트열로 나타낸다.
13 -> 00001101 -> 11110010 -> 11110011
5. byte 기반의 11001010 을 10진수 정수형으로 나타낸다.
00110110 
11001010 -> -54           
6. byte 기반의 11110101 을 10진수 정수형으로 나타낸다.
00001011
11110101 ->  -11
7. byte 기반의 10000001 을 10진수 정수형으로 나타낸다.
01111111
10000001 -> -127
